# Classic Tetris
## How to Run: 
From your command line, navigate to the "/Tetris" directory
```
$ cd /path/to/Tetris
```

Call pass gui.py to python 3.
```
$ python3 gui.py #on Linux
```
```
$ python gui.py #on Windows
```
## Current key bindings: 
w:"Rotate Right", s:"Drop", a:"Move Left", d:"Move Right"
